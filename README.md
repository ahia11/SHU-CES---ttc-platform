## 📖 项目简介

这是一个基于 **Vue.js 3** + **ECharts** 构建的交互式数据可视化分析系统。

本项目聚焦中国 **交通建设 (T)**、**旅游产业 (T)** 与 **居民消费 (C)** 三大核心领域，旨在将复杂、抽象的经济学模型（如空间溢出、向量自回归）与现代 Web 技术相结合。通过 **数据叙事** 与 **HCI（人机交互）创新**，将 2005-2023 年的多维面板数据转化为直观分析场景，揭示 T-T-C 关系从“单向拉动”到“双向互哺”的动态演化规律。


## ✨ 核心功能与 HCI 创新

本项目不仅是图表的堆砌，更是一套集成了多种 HCI 创新模式的分析工具，有效降低了复杂经济模型的认知门槛。

### 1. 🤖 "GUI + LUI" 双模式交互
为了降低使用门槛，系统集成了 **COZE 智能聊天机器人**。
* **GUI**: 传统的图形界面点击操作。
* **LUI**: 自然语言交互。用户可直接提问（例如：“四川省近十年的铁路客运量变化？”），实现“智慧咨询”。

### 2. 🔄 动态因果挖掘 (数据侦探)
通过“数据叙事”的设计理念，引导用户按 `宏观共振 -> 量化相关 -> 时序因果 -> 最终结论` 的逻辑层层深入。
* **交互式建模**：对比“早期 (2005-2014)”和“后期 (2015-2023)”的因果网络图，展示从“单向驱动”到“双向互哺”的质变。
* **参数化交互**：支持通过滑块实时调节“脉冲响应”冲击强度，即时演算并可视化结果。

### 3. 🔗 跨图表视图联动
解决多图表孤立问题，在“结构变迁”模块实现联动：
* **交互逻辑**：当鼠标悬停在 `Act 2 全国客运结构图` 的 Y 轴省份时，系统通过 `dispatchAction` 事件，自动高亮 `Act 3 铁路效率散点图` 中对应的省份点。
* **价值**：在“结构特征”与“效率水平”之间建立直接的视觉关联。

### 4. 🗺️ "概览 + 详情" 过滤模式
在“空间溢出”模块中解决流向图杂乱问题：
* **默认状态**：展示全国所有省份的模拟溢出关系总览。
* **交互过滤**：点击左侧地图任一省份（如江苏省），右侧流向图立刻过滤，仅显示“以江苏省为源头”的溢出关系，并更新统计卡片。

### 5. 💡 超级工具提示 (Super Tooltip)
在“多维数据探索”模块提升信息密度：
* **微型可视化**：鼠标悬停在热力图色块时，Tooltip 内通过 `nextTick` 异步渲染微型折线图 (Sparkline)，展示该省份完整的历史趋势，实现宏观与微观的瞬时切换。

## 🛠️ 技术栈

| 模块 | 技术选型 | 说明 |
| :--- | :--- | :--- |
| **核心框架** | Vue.js 3 | Composition API, Script Setup |
| **可视化库** | Apache ECharts 5 | 核心图表渲染 |
| **路由管理** | Vue Router 4 | 单页应用路由 |
| **状态管理** | Pinia | (如果使用了) 全局状态管理 |
| **UI 组件库** | Element Plus / AntD | (根据实际情况填写) 通用 UI 组件 |
| **AI 集成** | Coze API | 智能对话服务 |

## 🚀 快速开始

### 环境准备
确保你的本地环境已安装 [Node.js](https://nodejs.org/) (推荐 v16+)。

### 安装与运行

```bash
# 1. 克隆项目
git clone [https://github.com/ahia11/SHU-CES---ttc-platform.git]
# 2. 进入目录
cd TTC-Platform

# 3. 安装依赖
npm install
# 或者 yarn install / pnpm install

# 4. 启动开发服务器
npm run dev
